### 2. Server Setup Requirements
- Ensure server has Docker and Docker Compose installed
- Configure SSL certificates (Let's Encrypt via Certbot)
- Set up proper firewall rules (ports 80, 443)
- Configure database backups

### 3. Deployment Steps

1. **Set GitHub Secrets**
   ```bash
   # Example for setting secrets
   gh secret set DB_PASSWORD --body "your-secure-password"
   gh secret set SECRET_KEY --body "$(openssl rand -hex 32)"
   ```

2. **Test Build Locally**
   ```bash
   # Frontend
   cd frontend
   npm install
   npm run build
   
   # Backend
   cd backend
   pip install -r requirements.txt
   python -m pytest
   ```

3. **Run Quick Build Test**
   ```bash
   # Trigger the build test workflow
   gh workflow run build-test.yml
   ```

4. **Deploy to Production**
   ```bash
   # Push to main branch to trigger deployment
   git push origin main
   ```
